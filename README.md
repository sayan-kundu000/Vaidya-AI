# Vaidya AI

Vaidya AI is an AI-powered medical assistant web application built with a Flask backend. It leverages Google's Gemini AI to provide specialized conversational responses based on specialized medical personas and utilizes Google Text-to-Speech (gTTS) to read AI responses back to the user.

> **Disclaimer:** The information provided by Vaidya AI is generated by artificial intelligence and should not replace professional medical advice, diagnosis, or treatment. Always consult a certified healthcare professional before making any medical decisions or taking any medications.

## Features

- **Multi-Persona Medical AI:** Choose from a wide variety of specialist personas, including traditional medicine systems (Ayurveda, Siddha, Homeopathy) and comprehensive general practitioners (All-In-One Surgeon).
- **Gemini AI Integration:** Powered by `gemini-2.5-flash` model for high-quality, specialized responses.
- **Text-to-Speech (TTS):** Integrated audio playback system powered by `gTTS` to read the AI's responses out loud.
- **Mock Mode:** Operates safely in an offline/mock mode if no AI API key is configured.

## Tech Stack

- **Backend:** Python, Flask
- **AI Integration:** `google-genai`
- **Text-to-Speech:** `gTTS`
- **Environment Management:** `python-dotenv`
